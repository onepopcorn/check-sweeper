# Check-Sweeper
A clone of minesweeper video game but made with checkboxes

## Installation
Before anything be sure to have [gulp](http://http://gulpjs.com/) & [nodejs](htt://nodejs.org) so you need it to "compile" the project.

## Usage
Just type `gulp`
